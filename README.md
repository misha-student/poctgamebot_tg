🎮 Покемон Бот для Telegram
Легендарный бот для битв покемонов прямо в Telegram! Создавай, тренируй и сражайся со своими друзьями! ⚔️

📖 О проекте
Это Telegram бот, который позволяет играть в покемонов прямо в чате! Каждый игрок получает своего уникального покемона, может тренировать его, сражаться с друзьями и даже использовать секретные способности!

✨ Основные возможности
🎯 Создание покемона - каждый игрок получает случайного покемона из 151

⚡ Битвы - сражайся с друзьями в эпических битвах

📈 Система уровней - прокачивай своего покемона, побеждая врагов

🍕 Кормление - заботься о покемоне и получай награды

💊 Лечение - восстанавливай здоровье после битв

🏆 Достижения - получай награды за успехи

👑 Редкие покемоны - найди легендарного или мифического покемона!

🔥 Секретная атака - специальная команда для настоящих героев!

🚀 Установка и запуск
Шаг 1: Клонирование репозитория
bash

cd pokemon-bot
Шаг 2: Установка зависимостей
bash
pip install pyTelegramBotAPI requests
Шаг 3: Создание бота в Telegram
Найди @BotFather в Telegram

Отправь /newbot и следуй инструкциям

Получи токен бота

Шаг 4: Настройка
Создай файл config.py:

python
token = "ТВОЙ_ТОКЕН_БОТА"
Шаг 5: Запуск
bash
python main.py
📋 Команды бота
Команда	Описание
/go	Создать первого покемона
/new	Создать нового покемона
/info	Информация о твоем покемоне
/feed	Покормить покемона (раз в 20 часов)
/heal	Лечение покемона (раз в 10 сек)
/attack @username	Атаковать другого игрока
/top	Список всех игроков
/new10	Показать 10 случайных покемонов
/givepokemons	Раздать покемонов в группе
/SECRETATTACK100000POWER	💀 Секретная супер-атака!
